如何使用css3实现毛玻璃效果（背景模糊）

2021-05-21
  • css3

设计搞了一个背景模糊效果，一看内心有一万个mmp，但是这也难不倒我们这帮蓝领人才，头发没了也不全是加班呀！

filter

网上查了一下css3有一个filter的效果，可以实现背景模糊，上效果

上代码：

<!DOCTYPE html>
<html>

<head>
  <title>test</title>
  <style>
  html {
    overflow: hidden;
  }

  body {
    width: 100vw;
    height: 100vh;
    margin: 0;
    position: relative;
  }

  main {
    width: 50%;
    height: 50%;
    position: absolute;
    top: 0;
    left: 0;
    bottom: 0;
    right: 0;
    margin: auto;
    border-radius: 5px;
    overflow: hidden;
  }

  main::before {
    content: '';
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    /* filter */
    filter: blur(20px);
    z-index: -1;
    margin: -30px;
  }
  

  body,
  main::before {
    background: url("./girl.jpg") 0 / cover fixed;
  }
  </style>
</head>

<body>
  <main>
    <blockquote>
      "少年贪玩，青年迷恋爱情，壮年汲汲于成名成家，暮年自安于自欺欺人。人寿几何，顽铁能炼成的精金，能有多少？但不同程度的锻炼，必有不同程度的成绩；不同程度的纵欲放肆，必积下不同程度的顽劣。
      <br />
      <br />

      <footer>——
        <cite>杨绛</cite>
      </footer>
    </blockquote>
  </main>
</body>

</html>

很好，这很完美的实现了模糊的效果，但是有亿点点弊端，那就是main要加背景图片才可以实现模糊的效果，那如果父盒子没有背景图呢？或者这个父盒子是个地图、canvas、svg呢？

欸，恭喜，还真有解决的方法

backdrop-filter

欸，是不是一样的效果，很完美

<!DOCTYPE html>
<html>

<head>
  <title>test</title>
  <style>
  html {
    overflow: hidden;
  }

  body {
    width: 100vw;
    height: 100vh;
    margin: 0;
    position: relative;
    background: url("./girl.jpg") 0 / cover fixed;
  }

  main {
    width: 50%;
    height: 50%;
    position: absolute;
    top: 0;
    left: 0;
    bottom: 0;
    right: 0;
    margin: auto;
    border-radius: 5px;
    overflow: hidden;
    /* backdrop-filter */
    backdrop-filter: blur(20px);
  }
  </style>
</head>

<body>
  <main>
    <blockquote>
      "少年贪玩，青年迷恋爱情，壮年汲汲于成名成家，暮年自安于自欺欺人。人寿几何，顽铁能炼成的精金，能有多少？但不同程度的锻炼，必有不同程度的成绩；不同程度的纵欲放肆，必积下不同程度的顽劣。

      "上苍不会让所有幸福集中到某个人身上，得到爱情未必拥有金钱；拥有金钱未必得到快乐；得到快乐未必拥有健康；拥有健康未必一切都会如愿以偿。保持知足常乐的心态才是淬炼心智、净化心灵的最佳途径。一切快乐的享受都属于精神，这种快乐把忍受变为享受，是精神对于物质的胜利，这便是人生哲学。"
      <footer>——
        <cite>杨绛</cite>
      </footer>
    </blockquote>
  </main>
</body>

</html>
