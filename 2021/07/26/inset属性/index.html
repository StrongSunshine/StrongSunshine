inset属性

2021-07-26
  • css

不知道广大同胞们有没有在给元素定位写top、right、bottom、left写到自闭、虽然有scss函数、编辑器代码块等等方法。但是现在有了新方式啦。

inset

初一看，这不是boxShadow的属性吗？too young too naive。

inset这个属性是用来简化left/right/bottom/top这几个属性的，我们看看inset基本用法：

.element {
  position: absolute;
  inset: 0;
  /* 等同于 `top: 0; right: 0; bottom: 0; left: 0;` */
  inset: 1em 2em;
  /* 等同于 `top: 1em; right: 2em; bottom: 1em; left: 2em;` */
  inset: 1em 2em 3em;
  /* 等同于 `top: 1em; right: 2em; bottom: 3em; left: 2em;` */
  inset: 1em 2em 3em 4em;
  /* 等同于 `top: 1em; right: 2em; bottom: 3em; left: 4em;` */
}

inset-inline

相当于设置了left/right，需要注意的是，如果设置了writing-mode: vertical-lr，那么就会去设置top/bottom。
除了inset-inline，还有inset-inline-start和inset-inline-end，相当于分别设置这两个数值。（有卵用？）

inset-block

和inset-inline相反，相当于设置了top/bottom，如果设置了writing-mode: vertical-lr，那么就会去设置left/right。还有当然也有inset-block-start和inset-block-end，相当于分别设置这两个数值。（有卵用？）

注意：

使用之前需要caniuse一下哟，避免需要兼容浏览器，同时这个属性必须有设置position:relative或者position:absolute，否则没有效果。
