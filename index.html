css实现图片改变颜色

效果

参考： leafletjs官网–leafletjs

​ 鼠标悬停时改变叶子的颜色，不是切换图片，而是使用fillter的 hue-rotate属性，再配合animation实现渐变

image {
  animation: leafanim 3s ease 3s 1 normal none;
}

@keyframes leafanim {
    50% {
    -webkit-filter: hue-rotate(-70deg) saturate(1.2);
  }
}
